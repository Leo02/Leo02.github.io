今天玩了ifttt，发现可以用来做好多事，我先用[Rss-Bridge](https://bridge.suumitsu.eu/)获取了小狼ins的rss，然后设置同步到一个微博，然后用微博的rss同步到github的issue里，现在可以看了，我还把我和大雄的微博同步了，搞了大半天可以用了。

ifttt会把网站链接换成他们的短网址，那个短网址被屏蔽了，所以我申请了一个Gear免费空间，用curl获取短网址的location header，然后输出，在页面用jquery遍历文章内所有img标签，获取src，然后用replace获取短网址代码，在后端拼接成一个完整网站返回，jquery将原来的src替换为新src。这样做挺好的，就是点击的时候要顿一下，而且Gear的流量只有1GB/月，后来设置了一个缓存头，但是在本地测试不生效...在segmentfault问了大神，现在还没人理我。

#### JQuery：

    function gfw() {
    $(".content img").each(function(){
    var url=$(this).attr('src'); 
    var id=url.replace(/http:\/\/ift.tt\//, "");
    var replace=$.ajax({url:"http://zac.gear.host/?i="+id,async:false});
    $(this).attr('src',replace.responseText);      
    }); 
    }

#### Php：

    <?php
    header("Access-Control-Allow-Origin: *");
    header("Cache-Control: public");
    header("Pragma: cache");
    $offset = 30*60*60*24; // cache 1 month
    $ExpStr = "Expires: ".gmdate("D, d M Y H:i:s", time() + $offset)." GMT";
    header($ExpStr);
    $url = "http://ift.tt/".$_GET["i"];
    $ch=  curl_init($url);
    curl_setopt($ch, CURLOPT_FOLLOWLOCATION, 1);
    curl_setopt($ch, CURLOPT_AUTOREFERER, 1);
    curl_setopt($ch, CURLOPT_NOBODY, 1);
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
    curl_setopt($ch, CURLOPT_HEADER, 1);
    curl_exec($ch);
    $info = curl_getinfo($ch, CURLINFO_EFFECTIVE_URL);
    echo $info;
    ?>
